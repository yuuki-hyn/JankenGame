# じゃんけんゲーム
## 基本ルール
- プレイヤーとNPCはそれぞれ5つのライフを持つ
- ターン終了時にダメージ計算則に従ってライフを失う
- 相手のライフを0まで削った方の勝ち
- 最大10ターン
- 勝敗がつかずに10ターンを迎えた場合、ライフが多い方の勝利となる
- プレイヤーとNPCがじゃんけんを行う
- 同時に「攻撃」または「防御」を選択する
- じゃんけんの勝敗と攻守の選択によってダメージが決まる

## ダメージ計算
- (勝ち, 攻撃) - (負け, 攻撃) : 負けた方に2ダメージ
- (勝ち, 攻撃) - (負け, 防御) : 負けた方に1ダメージ
- (勝ち, 防御) - (負け, 攻撃) : 負けた方に1ダメージ
- (勝ち, 防御) - (負け, 防御) : 負けた方に0ダメージ

- (あいこ, 攻撃) - (あいこ, 攻撃) : 互いに1ダメージ
- (あいこ, 攻撃) - (あいこ, 防御) : 互いに0ダメージ
- (あいこ, 防御) - (あいこ, 防御) : 互いに0ダメージ
